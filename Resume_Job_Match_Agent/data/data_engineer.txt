JOB SUMMARY
------------------------------------------------------------

We are looking for an enthusiastic Data Engineer to build, maintain, and optimize
data pipelines that support analytics, reporting, and machine learning workloads.
The ideal candidate should have strong programming skills, database knowledge,
cloud fundamentals, and an understanding of modern data engineering tools.

------------------------------------------------------------
RESPONSIBILITIES
------------------------------------------------------------

- Design and develop scalable data pipelines.
- Build ETL/ELT workflows for structured and unstructured data.
- Write efficient SQL queries for data extraction and transformation.
- Integrate data from multiple sources.
- Perform data cleaning, preprocessing, and validation.
- Work with cloud-based data platforms.
- Support machine learning teams by preparing datasets.
- Monitor and optimize database performance.
- Collaborate with software developers, analysts, and data scientists.
- Follow Agile software development practices.
